## backpropagent

5년차 ML 엔지니어입니다. 모델을 설계하고 학습시키고 서빙하는 일을 해왔고, 지금은 LLM과 Agent 시스템으로 관심 영역을 넓히고 있습니다.

이 블로그에서는 다음과 같은 주제를 다룹니다.

- **LLM**: 파인튜닝, 프롬프팅, 서빙 최적화, 평가(evaluation)
- **Agent**: 멀티 에이전트 구조, 툴 사용(tool use), 오케스트레이션 프레임워크
- **ML 엔지니어링**: 그 외 모델링, 데이터 파이프라인, MLOps 관련 경험

ML 엔지니어로 쌓아온 기반 위에서, AI 엔지니어로서의 실험과 학습 과정을 기록으로 남기려 합니다.
